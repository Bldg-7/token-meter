TokenMeter — macOS widget for AI coding tools

Keep your Codex and Claude Code usage visible
on your desktop with native macOS widgets.

What it does:

> Widgets showing rate limits and remaining quota at a glance
> Tracks local token consumption over time
> Monitors multiple providers side by side
> Shows your current plan and reset timing

Usage data stays local. No external telemetry.
TokenMeter — AI 코딩 도구를 위한 macOS 위젯

Codex와 Claude Code의 사용량을
macOS 위젯으로 바탕화면에서 바로 확인하세요.

주요 기능:

> 플랜별 사용 한도와 남은 사용량을 위젯으로 표시
> 로컬 토큰 사용량을 시간별로 추적
> 여러 프로바이더를 한눈에 모니터링
> 현재 플랜과 리셋 시점 표시

사용 데이터는 로컬에만 저장됩니다. 외부 전송 없음.
TokenMeter — AIコーディングツール向けmacOSウィジェット

CodexとClaude Codeの使用量を
macOSウィジェットでデスクトップから直接確認できます。

主な機能:

> プランの使用上限と残量をウィジェットで表示
> ローカルのトークン消費量を時系列で追跡
> 複数のプロバイダーを一目でモニタリング
> 現在のプランとリセット時刻を表示

使用データはローカルにのみ保存されます。外部送信なし。

Visualiza el uso de Codex y Claude Code
directamente en tu escritorio con widgets nativos de macOS.
